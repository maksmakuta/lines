# Lines (WIP)

 Backend for Twitter/X like social network

### Features

- Bearer tokens authentification
- Simple pagination (in query)
- Put, get and delete any posts/comments/likes/subscriptions

### Dependencies

- Python 3.13+
- FastAPI
- SQLAlchemy
